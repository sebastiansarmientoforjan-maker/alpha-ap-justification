# ⚡ Quick Start: PDF Generation + Telegram

## 🎯 Objetivo
En 10 minutos tendrás un PDF profesional en tu Telegram.

---

## PASO 1: Instala LaTeX (5 minutos)

### Windows (tu sistema actual)

**Descarga MiKTeX:**
1. Ve a: https://miktex.org/download
2. Descarga "MiKTeX Installer" (32/64-bit según tu Windows)
3. Ejecuta el instalador
4. **IMPORTANTE:** En la instalación, selecciona:
   - ✅ "Install missing packages on-the-fly: **Yes**"
   - ✅ "For all users" (recomendado)
5. Espera que termine (~3 minutos)

**Verifica instalación:**
```bash
# Abre nueva terminal (Git Bash)
pdflatex --version

# Deberías ver:
# MiKTeX-pdfTeX 4.x (MiKTeX...)
```

Si da error "command not found", **reinicia la terminal** (Windows necesita reload del PATH).

## PASO 4: Genera tu Primer PDF (2 minutos)

```bash
# Asegúrate de que el servidor esté corriendo
npm run dev

# En OTRA terminal, ejecuta:
curl -X POST http://localhost:3000/api/admin/generate-frq \
  -H "Content-Type: application/json" \
  -d '{
    "studentId": "test-001",
    "studentName": "Ananya Kakarlapudi",
    "quizScore": 75,
    "weakTopics": ["derivatives", "MVT"],
    "course": "calculus-bc"
  }'
```

**¿Qué va a pasar?**

1. **Segundos 0-15:** Claude genera el FRQ completo
   - Problema
   - Solución CERC
   - Rúbrica CB
   - Justificación pedagógica
   - 3 problemas Week 1

2. **Segundos 15-20:** pdflatex compila el PDF
   - Genera archivo .tex
   - Compila 2 veces (para referencias)
   - Crea PDF final

3. **Segundos 20-25:** Se envía a Telegram
   - Sube el PDF como documento
   - Agrega botones inline

4. **Segundo 25:** 📱 DING en tu Telegram!

## 🐛 Troubleshooting Rápido

### ❌ Error: "pdflatex: command not found"

**Causa:** LaTeX no instalado o PATH no actualizado

**Solución:**
```bash
# Windows: Cierra y reabre la terminal
# Luego:
pdflatex --version

# Si sigue fallando, agrega manualmente al PATH:
# Busca la carpeta de instalación de MiKTeX:
# C:\Program Files\MiKTeX\miktex\bin\x64\
# Agrégala al PATH de Windows
```

### ❌ Error: "Package tcolorbox not found"

**Causa:** Paquetes LaTeX faltantes

**Solución (MiKTeX):**
1. Abre "MiKTeX Console"
2. Settings → General
3. Asegúrate: "Install missing packages on-the-fly: **Yes**"
4. Vuelve a ejecutar el comando

MiKTeX descargará paquetes automáticamente.

### ❌ Claude responde pero no se genera PDF

**Causa:** Error en compilación LaTeX

**Solución:** Revisa logs del servidor

```bash
# Busca en la consola donde corre npm run dev:
# Deberías ver:
📄 Generating FRQ PDF...
[pdflatex output]
✅ PDF generated: C:\Users\...\temp\pdfs\frq-test-001.pdf

# Si ves error, copia y pégalo para diagnóstico
```

### ❌ PDF se genera pero no llega a Telegram

**Causa:** Credenciales Telegram incorrectas

**Solución:**
```bash
# Verifica variables:
echo $TELEGRAM_BOT_TOKEN
echo $TELEGRAM_CHAT_ID

# Prueba envío directo:
curl http://localhost:3000/api/test-telegram?action=frq
```
